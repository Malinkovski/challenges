## Description

Challenge 14 jQuery, this challenge involves creating a race track using HTML, CSS and functionality implemented using jQuery. Local storage is utilized to store and retrieve results from previous races. The race track features a start button that initiates the race and resets the track, as well as a start over button that resets the race and positions the cars at the starting line. Also implemented a Clear history button to delete all race results and clear the local storage.
